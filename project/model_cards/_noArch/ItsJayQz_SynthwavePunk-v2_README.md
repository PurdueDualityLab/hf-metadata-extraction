### SynthwavePunk
This is a 50/50 Merge of Synthwave and InkPunk you can use both of their keywords and use prompt weighting to balance between these two cool and complimentary styles.

Synthwave token
'snthwve style'

Inkpunk token
'nvinkpunk'
