### Ghost in the Shell Diffusion

This is a fine-tuned Stable Diffusion model (based on v1.5) trained on 589 screen capture images taken from the **_Ghost in the shell Anime_**, . Use 
the token **_ghost style_** in your prompts to use the style.

[CKPT download link](https://huggingface.co/EOX/GhostintheShell/blob/main/ghost_style.ckpt)
